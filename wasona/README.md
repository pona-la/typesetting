# Wasona, a Toki Pona course

Here be a work-in-progress translation of Wasona markdown files to Typst.  
Currently just a couple lessons. The biggest roadblock is various custom html elements which need to be written from scratch in Typst. A smaller but still significant roadblock is adapting text to make sense in a printed context.
