# Typesetting

A hub for [Toki Pona](https://tokipona.net/) typesetting projects. We collaborate to produce PDFs of existing Toki Pona projects using Typst.

## Local development

* Install Typst.
* Run `make all`.
  * To compile just one project, use `make <dir-name>`.
  * If you don't have `make`, read the Makefile and run commands yourself.
